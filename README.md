# Texa - Text Smarter Privacy Policy & Terms

Modern ve responsive bir web sayfası olarak tasarlanmış Texa mobil uygulamasının gizlilik politikası ve kullanım şartları sayfası.

## 🚀 Özellikler

- **Modern Tasarım**: Gradient arka planlar ve modern UI/UX
- **Responsive**: Tüm cihazlarda mükemmel görünüm
- **Smooth Scrolling**: Yumuşak sayfa geçişleri
- **Interactive Elements**: Hover efektleri ve animasyonlar
- **Mobile Friendly**: Mobil cihazlar için optimize edilmiş
- **Accessibility**: Erişilebilirlik standartlarına uygun
