- **Mobile Friendly**: Mobil cihazlarda optimize edilmiş

## 🚀 Kurulum & Deployment

### Local Development
1. Projeyi klonlayın veya indirin
2. `index.html` dosyasını bir web tarayıcısında açın
3. Veya bir local server kullanın (örn: Live Server VS Code extension)

### GitHub Pages Deployment
1. Bu repository'yi GitHub'a push edin
2. Repository Settings > Pages
3. Source: Deploy from a branch
4. Branch: main (veya master)
5. Folder: / (root)
6. Save - otomatik olarak deploy edilecek

### App Store Kullanımı
- Privacy Policy URL: `https://[username].github.io/[repo-name]/#privacy`
- Terms URL: `https://[username].github.io/[repo-name]/#terms`
- Contact URL: `https://[username].github.io/[repo-name]/#contact`
